Q.Canvas and SVG tags

Ans:-

Canvas

  • Drawing is done with pixels
  • Animation are not built-in
  • High performance for pixel-bassed drawing operation
  • Not support for Event handler

SVG

  • Element are part of the page's DOM
  • Drawing is done with vectors
  • Animation are built-in
  • support for Event handler
